# Todo System

Hierarchical markdown structure. Obsidian-compatible.

## Structure

- `inbox.md` — unsorted tasks, quick captures
- `work/` — Proof IT related
- `personal/` — non-work
- `someday.md` — backlog, ideas, low priority

## Format

```markdown
- [ ] Task description
- [x] Completed task
```

Use `#tags` and `[[links]]` for cross-referencing.
